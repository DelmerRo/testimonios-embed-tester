# Test Embed - Testimonios CMS

Página para probar embeds de testimonios con HTTPS activo.

## Características
- ✅ HTTPS automático (Vercel)
- ✅ Sin problemas CORS
- ✅ Renderizado instantáneo
- ✅ Interfaz moderna

## Uso
1. Pega un ID de testimonio (UUID)
2. Obtén el código HTML
3. Pégalo y vélo renderizado

## API
`https://nocountry-cms.onrender.com/api/v1/public`
